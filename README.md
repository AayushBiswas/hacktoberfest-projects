## Run Locally

Clone the project

```bash
git clone https://github.com/max-programming/
```

Go to the project directory

```bash
cd hacktoberfest-projects
```

Install dependencies

```bash
yarn
# or npm Install
```

Start the server

```bash
yarn dev
# or npm run dev
```


## Tech Stack

Next.js, GitHub API, DaisyUI (Tailwind CSS)
